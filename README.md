# Pattern Tools
Libraries and tooling used within the Patterns of Life project.

## TypeScript npm packages
This repository contains the following two TypeScript libraries:

  - [patterns-ts](patterns-ts/) - source for `@patternsoflife/patterns`
  - [patterns-ts-cli](patterns-ts-cli/) - source for `@patternsoflife/patterns-cli`


## Python pypi packages
When we publish Python libraries, for example packages that could be imported
both in our web application backends and in our Jupyter notebooks, then those
should become part of this repository as well.
